```typescript
export interface ThemeListItem {
  id: string;
  name: string;
  slug: string;
  color: string;       // hex e.g. "#6366F1"
  emoji: string;       // e.g. "📚"
  contentCount: number;
  tags: { id: string; name: string }[];
  createdAt: string;
  updatedAt: string;
}

export interface ThemeRef {
  id: string;
  name: string;
  slug: string;
  color: string;
  emoji: string;
}
```

Do NOT add `themes` to the `Content` interface yet -- that's handled in Plan 07-02.

**2. Create `ios/hooks/useThemes.ts`** -- Follow the exact pattern from `useTopics.ts`. Create these hooks:

- `useThemes()` -- `GET /themes`, returns `ThemeListItem[]`. Query key: `['themes']`.
- `useThemeDetail(id, page?, platform?)` -- `GET /themes/:id?page=N&limit=20&platform=X`. Query key: `['themes', id, page, platform]`. `enabled: !!id`.
- `useCreateTheme()` -- `POST /themes` with body `{ name, color?, emoji? }`. Invalidates `['themes']`.
- `useUpdateTheme()` -- `PUT /themes/:id` with body `{ name?, color?, emoji? }`. Invalidates `['themes']` and `['themes', id]`.
- `useDeleteTheme()` -- `DELETE /themes/:id`. Invalidates `['themes']`.
- `useAddContentToTheme()` -- `POST /themes/:id/content` with body `{ contentIds }`. Invalidates `['themes']`, `['themes', themeId]`, `['content']`.
- `useRemoveContentFromTheme()` -- `DELETE /themes/:id/content/:contentId`. Invalidates `['themes']`, `['themes', themeId]`, `['content']`.

Use `api` from `../lib/api`. Import types from `../types/content`.

**3. Update `ios/hooks/index.ts`** -- Add export line:
```typescript
export { useThemes, useThemeDetail, useCreateTheme, useUpdateTheme, useDeleteTheme, useAddContentToTheme, useRemoveContentFromTheme } from './useThemes';
```
