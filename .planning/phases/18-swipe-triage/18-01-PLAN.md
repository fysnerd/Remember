    - Props: `onSwipeLeft: () => void`, `onSwipeRight: () => void`, `children: React.ReactNode`, `enabled?: boolean` (default true)
    - Import `Gesture`, `GestureDetector` from `react-native-gesture-handler`, `Animated`, `useSharedValue`, `useAnimatedStyle`, `withSpring`, `runOnJS`, `interpolate` from `react-native-reanimated`, `Dimensions` from `react-native`, `haptics` from `../../lib/haptics`.
    - Constants:
      - `SCREEN_WIDTH = Dimensions.get('window').width`
      - `SWIPE_THRESHOLD = SCREEN_WIDTH * 0.35` (35% of screen)
      - `VELOCITY_THRESHOLD = 500` (px/s fast flick)
      - `FLY_OFF_SPRING = { stiffness: 900, damping: 120, mass: 4 }` (fast, no bounce)
      - `SNAP_BACK_SPRING = { damping: 15, stiffness: 150, mass: 1 }` (bouncy, satisfying)
    - Use `useSharedValue(0)` for `translateX` and `translateY`.
    - Track if swipe threshold was crossed with a ref (`hasTriggeredHaptic`) to fire `haptics.medium()` once when crossing threshold during drag (in `onUpdate`).
    - Create `Gesture.Pan()`:
      - `.enabled(enabled)` -- disable when not top card
      - `.onUpdate((event)` => set translateX to event.translationX, translateY to `event.translationY * 0.3` (dampen vertical). Check if |translationX| > SWIPE_THRESHOLD and haptic not yet triggered -> `runOnJS(haptics.medium)()` and set ref.
      - `.onEnd((event)` => Determine direction:
        - `shouldSwipeRight`: `translateX.value > SWIPE_THRESHOLD || event.velocityX > VELOCITY_THRESHOLD`
        - `shouldSwipeLeft`: `translateX.value < -SWIPE_THRESHOLD || event.velocityX < -VELOCITY_THRESHOLD`
        - If right: `translateX.value = withSpring(SCREEN_WIDTH * 1.5, FLY_OFF_SPRING, (finished) => { if (finished) runOnJS(onSwipeRight)() })`, fire `runOnJS(haptics.success)()`
        - If left: `translateX.value = withSpring(-SCREEN_WIDTH * 1.5, FLY_OFF_SPRING, (finished) => { if (finished) runOnJS(onSwipeLeft)() })`, fire `runOnJS(haptics.warning)()`
        - Else: snap back both X and Y with SNAP_BACK_SPRING, reset haptic ref.
    - `cardStyle = useAnimatedStyle` with transform: translateX, translateY, rotate (interpolate translateX from [-SCREEN_WIDTH/2, 0, SCREEN_WIDTH/2] to [-10, 0, 10] degrees).
    - Render: `<GestureDetector gesture={panGesture}><Animated.View style={[styles.cardContainer, cardStyle]}>{children}<SwipeOverlay translateX={translateX} /></Animated.View></GestureDetector>`
    - Export `translateX` shared value is NOT needed externally -- SwipeOverlay is rendered INSIDE SwipeCard.
    - Style `cardContainer`: `{ width: '100%' }`
